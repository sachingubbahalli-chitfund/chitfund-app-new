# Basaveshwara Co-operative Society Chit Fund App

This project is a transparent, mobile-friendly chit fund management system built with **React** and **Firebase**.  
It is designed to help community members in Gubbahalli track contributions, loans, and collections with clarity.

## ✨ Features
- Member registration and authentication (Firebase Auth)
- Dashboard with society name and welcome message
- Members list section
- Collections tracking section
- Settings for app preferences
- Firebase Firestore integration for real-time updates

## 📂 Project Structure
- `firebase.js` → Firebase configuration
- `Auth.js` → User authentication functions
- `Dashboard.js` → Main dashboard view
- `Members.js` → Members listing
- `Collections.js` → Contributions and collections
- `Settings.js` → App preferences
- `App.js` → Main app structure
- `index.js` → Entry point

## 🚀 Getting Started
1. Clone the repository  
2. Run `npm install`  
3. Add your Firebase project configuration in `firebase.js`  
4. Run `npm start` to launch the app  

## 📖 Purpose
This app empowers rural communities with transparent chit fund tracking, bridging urban tech solutions with rural accessibility.  
It is tailored for **Basaveshwara Co-operative Society, Gubbahalli**.
